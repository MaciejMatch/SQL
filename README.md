# SQL :wrench:

This repository contains sample databases and SQL queries for testing a tester's SQL skills.

## Setup
1. Start your database (MySQL/PostgreSQL).
2. Load the `setup/sample_database.sql` file.

## Folder Structure
- `setup/` - scripts to create the database
- `queries/` - sample SQL queries
- `tests/` - test cases and queries for verification
- `docs/` - instructions for testers
